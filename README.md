# 📚 Guia de Tags HTML

## Estrutura Básica e Formatação de Texto

| Tag               | Descrição                                                   |
| ----------------- | ----------------------------------------------------------- |
| `<html>`          | Envolve e inicia todo o código da página.                   |
| `<head>`          | Guarda as configurações e metadados invisíveis da página.   |
| `<title>`         | Define o título que aparece na aba do navegador.            |
| `<meta>`          | Configura informações gerais (como idioma e acentuação).    |
| `<body>`          | Contém todo o conteúdo visível do site.                     |
| `<h1>` até `<h6>` | Títulos e subtítulos (do principal `<h1>` ao menor `<h6>`). |
| `<p>`             | Cria um parágrafo de texto comum.                           |
| `<br>`            | Quebra a linha, jogando o texto seguinte para baixo.        |
| `<hr>`            | Desenha uma linha horizontal para separar conteúdos.        |
| `<strong>`        | Destaca o texto em negrito (indica importância).            |
| `<em>`            | Destaca o texto em itálico (indica ênfase na leitura).      |
| `<mark>`          | Destaca o texto com um fundo colorido (efeito marca-texto). |
| `<small>`         | Deixa o texto com a fonte menor (avisos, notas de rodapé).  |
| `<sub>`           | Texto subscrito (posicionado abaixo da linha normal).       |
| `<sup>`           | Texto sobrescrito (posicionado acima da linha normal).      |

---

## Listas

| Tag    | Descrição                                        |
| ------ | ------------------------------------------------ |
| `<ol>` | Abre uma lista ordenada (com números ou letras). |
| `<ul>` | Abre uma lista não ordenada (com marcadores).    |
| `<li>` | Define cada item individual dentro de uma lista. |

---

## Estrutura Semântica e Artigos

| Tag            | Descrição                                                            |
| -------------- | -------------------------------------------------------------------- |
| `<header>`     | Cabeçalho principal da página ou de uma seção.                       |
| `<main>`       | Define onde fica o conteúdo principal e exclusivo da página.         |
| `<section>`    | Divide a página em seções ou blocos de assuntos diferentes.          |
| `<article>`    | Bloco de conteúdo independente (ex.: notícia, palestra, postagem).   |
| `<time>`       | Identifica datas e horários de forma compreensível para o navegador. |
| `<address>`    | Informações de contato do autor, palestrante ou empresa.             |
| `<blockquote>` | Bloco para citações longas.                                          |
| `<cite>`       | Indica a fonte ou o autor de uma citação.                            |

---

## Tabelas

| Tag         | Descrição                           |
| ----------- | ----------------------------------- |
| `<table>`   | Cria a estrutura da tabela.         |
| `<caption>` | Título ou legenda da tabela.        |
| `<thead>`   | Agrupa os títulos das colunas.      |
| `<tbody>`   | Agrupa o corpo principal da tabela. |
| `<tfoot>`   | Agrupa resumos ou totais no rodapé. |
| `<tr>`      | Cria uma linha na tabela.           |
| `<th>`      | Cria uma célula de cabeçalho.       |
| `<td>`      | Cria uma célula comum de dado.      |

---

## Mídia e Interação

| Tag            | Descrição                                       |
| -------------- | ----------------------------------------------- |
| `<figure>`     | Agrupa uma imagem ou gráfico com sua legenda.   |
| `<figcaption>` | Legenda da imagem dentro de `<figure>`.         |
| `<img>`        | Insere uma imagem na página.                    |
| `<video>`      | Cria um player de vídeo.                        |
| `<audio>`      | Cria um player de áudio.                        |
| `<source>`     | Define o caminho e formato do arquivo de mídia. |
| `<details>`    | Cria uma caixa retrátil (accordion).            |
| `<summary>`    | Título clicável de um elemento `<details>`.     |

---

## Formulários

| Tag          | Descrição                                       |
| ------------ | ----------------------------------------------- |
| `<form>`     | Cria a área do formulário.                      |
| `<fieldset>` | Agrupa campos relacionados dentro de uma borda. |
| `<legend>`   | Define o título do `<fieldset>`.                |
| `<label>`    | Rótulo associado a um campo de formulário.      |
| `<textarea>` | Caixa de texto para mensagens longas.           |
| `<select>`   | Cria um menu suspenso (dropdown).               |
| `<option>`   | Define uma opção dentro de `<select>`.          |
| `<button>`   | Cria um botão clicável.                         |
| `<input>`    | Campo básico para entrada de dados.             |

### Tipos do `<input>`

| Tipo       | Descrição                                          |
| ---------- | -------------------------------------------------- |
| `text`     | Caixa simples para textos curtos.                  |
| `email`    | Valida endereços de e-mail.                        |
| `password` | Oculta os caracteres digitados.                    |
| `date`     | Exibe um calendário para seleção de data.          |
| `number`   | Permite apenas números.                            |
| `file`     | Permite o envio de arquivos.                       |
| `radio`    | Seleção única entre várias opções.                 |
| `checkbox` | Permite múltiplas seleções.                        |
| `tel`      | Campo otimizado para telefones.                    |
| `url`      | Valida endereços de sites.                         |
| `color`    | Exibe uma paleta de cores.                         |
| `range`    | Cria uma barra deslizante para selecionar valores. |

---

## 📖 Resumo

Este guia reúne as principais tags HTML utilizadas para:

* Estruturação de páginas
* Formatação de textos
* Criação de listas
* Organização semântica
* Construção de tabelas
* Inserção de mídias
* Desenvolvimento de formulários

Ideal para consultas rápidas durante o aprendizado de HTML.
